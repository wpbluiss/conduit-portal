# Conduit AI — Customer Portal

> Web-based portal for Conduit AI customers to manage their AI voice agents, leads, and billing.

**Live:** [app.conduitai.io](https://app.conduitai.io)

## Overview

The Conduit AI portal is the customer-facing dashboard where service businesses manage their AI voice agent, view captured leads, track analytics, configure settings, and handle billing. Built with a mobile-responsive design featuring hamburger navigation and stacked layouts.

## Tech Stack

- **Frontend:** React, JavaScript, Tailwind CSS
- **Backend API:** Python, FastAPI (deployed on Railway)
- **Database:** Supabase (PostgreSQL)
- **Authentication:** Supabase Auth
- **Payments:** Stripe (subscriptions, webhooks, billing portal)

## Features

- **Dashboard** — Real-time call volume, lead metrics, conversion analytics
- **Lead Management** — Full lead details, call transcripts, contact info
- **AI Agent Settings** — Configure voice agent greeting, behavior, business info
- **Business Settings** — Company profile, hours, service area
- **Integrations** — Zapier, Google Calendar sync, Booksy
- **Referrals** — Affiliate system with unique codes and commission tracking
- **Billing** — Stripe-powered subscription management
- **Mobile Responsive** — Hamburger menu, stacked layouts for all screen sizes
